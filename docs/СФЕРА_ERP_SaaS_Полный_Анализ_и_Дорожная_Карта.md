## 📖 История проекта (Откуда мы пришли)

### Версия 1.0 — ЛЕОНИКА CRM (Локальная, отраслевая, АКЗ)

Внутренний инструмент строительной компании ООО ЛЕОНИКА, специализирующейся на
антикоррозийной защите (АКЗ), пескоструйной обработке и огнезащите.

**Стек V1:** React + TypeScript (Vite) → FastAPI → SQLite (WAL mode) → SSH-туннель → VPS Timeweb → Nginx + Let's Encrypt

**Что было построено и работает (≈ 30+ завершённых задач):**

| Модуль | Что сделано |
|---|---|
| **Ядро** | Kanban-доска объектов, задачи с Drag&Drop, склад, оборудование, финансы, клиенты |
| **Безопасность** | JWT + HttpOnly cookies, rate limiting, XSS-защита |
| **Инфраструктура** | Поддомен срм.леоника56.рф, SSH reverse tunnel (Paramiko + Timeweb VPS) |
| **Тендеры** | Парсер zakupki.gov.ru (live RSS) + 11 коммерческих площадок + AI-фильтр + AI-анализ ТЗ |
| **AI-Копилот** | Ollama qwen2:7b в чате и Telegram, AI-анализ договоров, саммаризация ТЗ |
| **Голос** | Voice-to-Lead (Telegram → Groq Whisper → Ollama → авто-создание Client+Task) |
| **B2B Парсер** | ОКВЭД-парсер (api-fns.ru → rusprofile → AI-скоринг), Снайпер-парсер (DuckDuckGo) |
| **Telegram** | Бот (long polling), Mini App (glassmorphism, haptic feedback), inline-кнопки, треды |
| **Документы** | Генерация КП/счетов в Docx, PDF-парсинг документов тендеров |
| **Система** | Audit Trail с диффом, Ops Monitor (6 проверок), Decision Log, утренний дайджест |

**Статус:** ✅ Активна, работает онлайн на `срм.леоника56.рф` (порт 8000)

### Версия 2.0 — СФЕРА ERP SaaS (Коммерческая, мультитенантная)

**Событие сепарации:** Рабочая CRM заархивирована/скопирована. Новая ветка изолирована:
- Backend: порт `8001` (локально)
- Frontend: порт `5173` (локально, Vite dev server)
- База данных: **Neon PostgreSQL** (Cloud Serverless, с RLS)
- Старая CRM: продолжает работать на порту 8000, не затрагивается

**Что уже реализовано в СФЕРЕ (SaaS-слой):**
- Мультитенантное ядро на PostgreSQL Neon с `tenant_id` + SQLAlchemy RLS-фильтр (ContextVar)
- B2B-регистрация по ИНН с автозаполнением из ФНС (api-fns.ru)
- Биллинг: статус лицензии, генерация счетов Word Docx, история счетов, тестовая оплата
- `database.py` — глобальный RLS через `before_compile` hook SQLAlchemy

## 💡 Концепция продукта

> **СФЕРА ERP** — это **B2B-Экосистема с ИИ**, где:
> - Встроенный AI-парсер тендеров и ОКВЭД-парсер **сами приносят лиды**
> - Внутренний Маркетплейс даёт **реальные заказы** от других компаний
> - Отраслевые модули помогают **выполнить** эти заказы
> - AI-агенты автоматизируют рутину, оставляя людям ключевые решения

**УТП:** Ни amoCRM, ни Битрикс24 не умеют **сами находить клиентов и тендеры** через встроенный AI-парсер.
